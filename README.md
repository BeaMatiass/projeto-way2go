1. Vejo onde estou.
2. Vejo onde quero chegar.
3. Calculo a distância até o alvo.
4. Calculo para qual direção o robô deveria apontar.
5. Comparo essa direção com o yaw atual.
6. Se estiver longe e torto, gira parado.
7. Se estiver longe e mais ou menos apontado, anda.
8. Se chegou na posição, para de andar e ajusta o yaw final.
9. Se chegou na posição e no yaw, para tudo.
